#1-2. Visual Studio Code

개발자가 되기위해서, 우선 텍스트 에디터가 필요합니다. 웹사이트는 텍스트의 나열이기 때문에, 이론적으로는 메모장만을 사용해서 사이트를 만들 수 있습니다. 다른 코딩작업도 마찬가지고요.

하지만 우리는 코드를 작성할 때 가독성을 높히기 위해 색깔을 자동으로 입혀준다던가, 문법적으로 잘못된 부분을 알려주는 등 우리의 코딩을 편하게 하게 해줄 코딩 전용 텍스트 에디터를 사용할 겁니다.

텍스트 에디터의 종류는 많습니다. 그 중에서 우리는 현재 가장 유행하는 에디터 중 하나인 'Visual Studio Code'를 사용할 것입니다.
